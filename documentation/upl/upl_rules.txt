 сезон
-----------------------------------------------------------------------------------------------------------------

# каждый сезон состоит из нескольких турниров:
	# чемпионат украины:
		# 2 группы по 8 клубов в каждой (перваЯ лига/втораЯ лига).
		# олимпийскаЯ система встреч (по 1 очной встрече, 7 туров, 
		  по 4 матча в туре длЯ каждой лиги).
		# стартовые места в таблице определЯютсЯ результатами предыдущего 
		  сезона, длЯ первого сезона - датами основаниЯ клубов.
		# по результатам сезона 2 последних клуба первой лиги вылетают вниз 
		  во вторую лигу, а на их место из второй лиги подымаютсЯ 2 первых клуба.
		# при прочих равных, выше занимает позицию тот клуб, у которого лучше 
		  показатель (в порЯдке приоритетности):
			# набранных очков;
			# очной встречи с соперником;
			# разница забитых-пропущенных голов;
			# стоимость клуба на момент начала турнира.
		# клуб-победитель получает 50 копеек к стоимости.
	# кубок украины:
		# кубок из всех 16 клубов чемпионата.
		# олимпийскаЯ система встреч (4 тура по 1 матчу).
		# стартовые места в сетке определЯютсЯ полуслепым жребием - клубы из 
		  одной лиги не могут перечечьсЯ в первом туре.
		# клуб-победитель получает 25 копеек к стоимости.
	# кубок гетьмана хмельницкого:
		# кубок из 8 клубов первой лиги.
		# олимпийскаЯ система встреч (3 тура по 1 матчу).
		# стартовые места в сетке определЯютсЯ слепым жребием.
		# клуб-победитель получает 10 копеек к стоимости.
	# суперкубок украины:
		# матч между обладателем кубка украины и чемпионом украины.
		# клуб-победитель получает 5 копеек к стоимости.

# по окончанию сезона подсчитываетсЯ статистика:
	# каждый клуб получает по 1 копейке к стоимости за каждую победу.
	# обладатели трофеев получают стоимость выиграных ими трофеев.
	# обновлЯетсЯ график выигранных трофеев.



-----------------------------------------------------------------------------------------------------------------
 матч
-----------------------------------------------------------------------------------------------------------------

# длЯ каждого матча:
	# атака 1й команды.
	# атака 2й команды.
	# забег капитана 1й команды.
	# забег капитана 2й команды.
	# если ничьЯ и матч не групповой, тогда бить послематчевые 
	  пенальки (пробивают до первого незабитого).
	
# длЯ каждой атаки:
    # тренер атакующего клуба бросает кубики.
	# сумма выпавших значений = забитые голы.
	# если уровень атакующего клуба выше чем у соперника, тогда	
	  тренер может 1 раз перебросить кубики (обЯзательно оба).
	# учитываютсЯ только переброшеные значениЯ.

# длЯ каждого пенальки:
	# тренер атакующего клуба бросает кубики.
	# тренер оборонЯющегосЯ клуба бросает кубики.
	# если удар > броска вратарЯ, тогда гол.

# длЯ каждого забега капитана:
    # тренер атакующего клуба бросает кубики.
	# если сумма выпавших значений = 4, тогда гол и еще 
	  один забег того же капитана.
	# если сумма выпавших значений != 4, тогда забег 
	  капитана окончен.
